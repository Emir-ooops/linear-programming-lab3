# Решение методом множителей Лагранжа

## 1. Составление функции Лагранжа

Вводим два множителя Лагранжа: λ1 и λ2.

Функция Лагранжа:

L(x, y, λ1, λ2) =
3x² + 2xy + 2y²

* λ1(x + 2y − 2)
* λ2(2x − y − 1)

---

## 2. Условия стационарности

Находим частные производные и приравниваем их к нулю.

Производная по x:

dL/dx = 6x + 2y + λ1 + 2λ2 = 0

Производная по y:

dL/dy = 2x + 4y + 2λ1 − λ2 = 0

Производная по λ1 возвращает первое ограничение:

x + 2y − 2 = 0

Производная по λ2 возвращает второе ограничение:

2x − y − 1 = 0

Получаем систему из четырёх уравнений с четырьмя неизвестными.

---

## 3. Решение системы ограничений

Рассмотрим систему:

x + 2y = 2
2x − y = 1

Из второго уравнения:

y = 2x − 1

Подставляем в первое:

x + 2(2x − 1) = 2
x + 4x − 2 = 2
5x = 4
x = 0.8

Тогда:

y = 2 · 0.8 − 1 = 0.6

Допустимое множество состоит из единственной точки:

(x*, y*) = (0.8, 0.6)

Так как в R² заданы два независимых линейных равенства, они определяют пересечение двух прямых, то есть одну точку.

---

## 4. Нахождение множителей Лагранжа

Подставляем x = 0.8 и y = 0.6 в уравнения стационарности.

Из dL/dx:

6·0.8 + 2·0.6 + λ1 + 2λ2 = 0
4.8 + 1.2 + λ1 + 2λ2 = 0
λ1 + 2λ2 = −6

Из dL/dy:

2·0.8 + 4·0.6 + 2λ1 − λ2 = 0
1.6 + 2.4 + 2λ1 − λ2 = 0
2λ1 − λ2 = −4

Решая систему:

λ1 = −2.8
λ2 = −1.6

---

## 5. Значение целевой функции

f(0.8, 0.6) =
3·0.8² + 2·0.8·0.6 + 2·0.6²

= 3·0.64 + 2·0.48 + 2·0.36
= 1.92 + 0.96 + 0.72
= 3.6

# Проверка характера точки (критерий Сильвестра)

Матрица Гессе функции f(x, y):

H =
[ 6   2 ]
[ 2   4 ]

Главные миноры:

Δ1 = 6 > 0
Δ2 = 6·4 − 2·2 = 24 − 4 = 20 > 0

Так как оба главных минора положительны, матрица положительно определена.

Следовательно, функция f(x, y) строго выпуклая.

При линейных ограничениях это означает, что найденная точка соответствует условному минимуму.

Кроме того, поскольку допустимое множество состоит из одной точки, она автоматически является минимумом на этом множестве.
